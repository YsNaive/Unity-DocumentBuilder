Export as Markdown

To export a book into markdown format :

goto " Edit / Project Settings / DocumentBuilder Settings " and see " Esport " section like below

  • Path is export root folder
  • root is the book root, notice that all of its subPages will export at the same time

After generated, it will also generate a SUMMARY.md file it contains docs struct and releated-path link, you can use it to export static site with using other html tool like GitBook.
